# Task 9.1 Protocols and Ports v2
**What is the purpose of the following protocols and which ports do they use?**
1. SSH
   Port     : 22
   Purpose  : To encrypt and securely log in to and execute commands remotely on computers on a network.

2. HTTPS
   Port     : 443
   Purpose  : To encrypt communication between the web browser and the web server.

3. Telnet
   Port     : 23
   Purpose  : To remotely log in to computers on a network. This is an older protocol.

4. DNS
   Port     : 53
   Purpose  : To translate domain names into IP addresses.

# Task 9.2. A Quick Quiz
**Answer the following questions and explain your answers:**
1. Can you use sockets to build a P2P system?
   Answer : Yes
   Reason : In P2P, each host acts as both a client and a server.

2. Can a computer have several associated IP addresses?
   Answer : Yes
   Reason : A computer can have multiple IP addresses and DNS names.

3. Can a computer maintain several socket connections established viathe same portnumber?
   Answer : Yes
   Reason : A server can listen on a particular port and accept connections from multiple clients, because the operating system uses a unique identifier for each connection to distinguish between them even after the connection is established.

4. Suppose you have sent three UDP packets: p1, then p2, then p3. Will they arrive in the same order?
   Answer : No
   Reason : UDP is a connectionless protocol and does not guarantee the order of arrival.
